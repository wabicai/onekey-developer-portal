# 传输层概览

传输层（Transport）是应用程序与 OneKey 硬件设备之间的通信层。根据你的平台和连接方式选择合适的传输方案。

> 如果下面的标准传输方式都无法满足（例如需要自定义硬件通道），再查看文末的「高级主题」并考虑底层传输插件/消息协议。

## 连接方式

OneKey 设备支持两种主要连接方式：

| 方式 | 连接类型 | 支持设备 |
|-----|---------|---------|
| **USB** | 有线 USB 连接 | 所有 OneKey 设备 |
| **蓝牙** | 无线 BLE 连接 | OneKey Pro、OneKey Touch |

## 选择传输方案

根据下表选择适合你项目的传输方案：

| 平台 | 推荐方案 | 包名 |
|-----|---------|-----|
| **Web 浏览器** | [WebUSB](./web-usb) | `@onekeyfe/hd-common-connect-sdk` |
| **React Native** | [React Native BLE](./react-native-ble) | `@onekeyfe/hd-ble-sdk` |
| **Android (原生)** | [Android 原生](./android) | 底层传输插件 |
| **iOS (原生)** | [iOS 原生](./ios) | 底层传输插件 |
| **Flutter** | [Flutter 原生](./flutter) | 底层传输插件 |

## 架构

```
┌─────────────────────────────────────────────────────────┐
│                      你的应用程序                         │
├─────────────────────────────────────────────────────────┤
│                  Hardware SDK (JavaScript)               │
├──────────────────┬──────────────────┬───────────────────┤
│     WebUSB       │  React Native    │    底层传输插件    │
│     (Web)        │      BLE         │     (Native)      │
├──────────────────┴──────────────────┴───────────────────┤
│                   USB  /  蓝牙 BLE                        │
└─────────────────────────────────────────────────────────┘
```

## Web 开发

### WebUSB

适用于 Chrome/Edge 浏览器中运行的 Web 应用。通过 USB 线缆连接。

- 需要 HTTPS
- 需要用户手势授权设备
- 无需原生依赖

[开始使用 WebUSB](./web-usb)

## 移动端开发

### React Native BLE

适用于 React Native 应用。使用 `@onekeyfe/hd-ble-sdk` 包和原生 BLE 模块。

- 纯 JavaScript SDK，原生传输层
- 支持 iOS 和 Android
- 需要 Dev Client 构建（非 Expo Go）

[开始使用 React Native BLE](./react-native-ble)

### 原生平台 (Android / iOS / Flutter)

适用于不使用 React Native 的原生应用。这些页面都是 **基于底层传输插件的示例实现**，通过 WebView + JS Bridge + 原生 BLE 适配。请先阅读《底层传输插件》，再参考对应示例。

架构：**WebView + JS Bridge + 原生 BLE**

| 平台 | 指南 | 示例 |
|-----|-----|-----|
| Android | [Android 原生](./android) | [native-android-example](https://github.com/OneKeyHQ/hardware-js-sdk/tree/onekey/packages/connect-examples/native-android-example) |
| iOS | [iOS 原生](./ios) | [native-ios-example](https://github.com/OneKeyHQ/hardware-js-sdk/tree/onekey/packages/connect-examples/native-ios-example) |
| Flutter | [Flutter 原生](./flutter) | - |
