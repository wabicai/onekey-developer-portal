# deviceVerify

## 验证设备

从设备获取硬件签名信息的 cert 和 signature 字段，并提交到 OneKey 服务器验证设备的可靠性。读取签名数据需要在设备上确认。

```typescript
const result = await HardwareSDK.deviceVerify(connectId, params);
```

## 参数

[**可选通用参数**](../api-reference/common-params)


* `dataHex` - _可选_ `string` 十六进制字符串形式的随机字符串，例如以十六进制字符串传入的时间戳。

## 示例

```typescript
HardwareSDK.deviceVerify(connectId, {
    dataHex: '183533a058f'
});
```

## 返回结果

```typescript
{
    success: true,
    payload: {
        cert: string
        signature: string
    }
}
```

错误

```typescript
{
    success: false,
    payload: {
        error: string, // 错误信息
        code: number // 错误码
    }
}
```
