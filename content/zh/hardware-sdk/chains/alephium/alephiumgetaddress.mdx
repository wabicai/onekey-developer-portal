# alephiumGetAddress

### 使用要求

* 固件版本要求
  * Touch: 4.10.0

### Alephium：获取地址

在设备上显示由给定 BIP32 路径派生的请求地址，并将其返回给调用者。用户需要在 OneKey 上确认导出。

```typescript
const result = await HardwareSDK.alephiumGetAddress(connectId, deviceId, {
    path: "m/44'/1234'/0'/0/0",
    showOnOneKey: false,
    group: 0,
    includePublicKey: true
});
```

#### 参数

[可选通用参数](../../../api-reference#common-params)

**导出单个地址**

* `path` - _必需_ `string | Array<number>` 最小长度为 3
* `group` - _可选_ `number | undefined | null`
* `showOnOneKey` - _可选_ `boolean` 决定是否在设备上显示地址。默认设置为 `true`

**导出批量地址**

* `bundle` - 包含 `path`、`showOnOneKey` 和 `group` 字段的对象 `Array`

#### 示例

```typescript
// 获取单个地址
const response = await HardwareSDK.alephiumGetAddress(connectId, deviceId, {
    path: "m/44'/1234'/0'/0/0",
    showOnOneKey: true
});

// 获取多个地址
const batchResponse = await HardwareSDK.alephiumGetAddress(connectId, deviceId, {
    bundle: [
        {
            path: "m/44'/1234'/0'/0/0",
            showOnOneKey: false
        },
        {
            path: "m/44'/1234'/1'/0/0",
            showOnOneKey: false
        }
    ]
});
```

**返回结果**

仅返回一个地址的结果

```typescript
{
    success: true,
    payload: {
        address: string,     // 显示的地址
        derivedPath: string,
        publicKey: string,
        path: Array<number> // 强化路径
    }
}
```

返回批量地址的结果（按 FIFO 排序）

```typescript
{
    success: true,
    payload: [
        { address: string, derivedPath: string, publicKey: string, path: Array<number> }, // 账户 1
        { address: string, derivedPath: string, publicKey: string, path: Array<number> }, // 账户 2
        { address: string, derivedPath: string, publicKey: string, path: Array<number> }  // 账户 3
    ]
}
```

错误

```
{
    success: false,
    payload: {
        error: string, // 错误信息
        code: number // 错误码
    }
}
```
