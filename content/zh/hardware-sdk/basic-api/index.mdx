# 基础 API

用于设备发现、会话管理和事件处理的核心 SDK 方法。

| 方法 | 描述 |
|------|------|
| [Search Devices](./search-devices) | 扫描并发现已连接的 OneKey 硬件设备 |
| [Get Features](./get-features) | 获取设备能力和当前状态 |
| [Get Passphrase State](./get-passphrase-state) | 获取当前密语钱包状态 |
| [Response UI Event](./response-ui-event) | 处理来自设备交互的 UI 事件 |
| [Cancel Request](./cancel-request) | 取消正在进行的设备操作 |
