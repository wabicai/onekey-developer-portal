# nostrSignSchnorr

### 使用要求

* 固件版本要求
  * Touch: 4.6.0
  * Classic/Mini: 3.5.0
  * Classic1s: 3.6.0

## Nostr：签名 Schnorr

请求设备使用由给定 BIP32 路径派生的私钥对给定的 Schnorr 交易进行签名。用户需要在 OneKey 上确认所有交易详情。

```typescript
const result = await HardwareSDK.nostrSignSchnorr(connectId, deviceId, params);
```

### 参数

[**可选通用参数**](../common-params)

* `path` - _必需_ `string | Array<number>` 最小长度为 `3`。[更多信息](../path)
* `hash` - _必需_ `string` Schnorr 哈希。



### 示例

```typescript
const response = await HardwareSDK.nostrSignSchnorr(
  connectId,
  deviceId,
  {
    path: "m/44'/1237'/0'/0/0",
    hash: '2118c65161c7d68b4bdbe1374f658532670057ab1bb0c99937d0ff7cff45cb5e',
  }
);
```

返回结果

```typescript
{
  success: true,
  payload: {
    path: string;
    rawHash: string,
    signature: string,
  }
}
```

错误

```typescript
{
    success: false,
    payload: {
        error: string, // 错误信息
        code: number // 错误码
    }
}
```
