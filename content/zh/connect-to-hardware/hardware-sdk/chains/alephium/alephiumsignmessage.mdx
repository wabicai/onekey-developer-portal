# alephiumSignMessage

### 使用要求

* 固件版本要求
  * Touch: 4.10.0

### Alephium：签名消息

使用由给定 BIP32 路径派生的私钥对消息进行签名。用户需要在 OneKey 设备上确认操作。

```typescript
// 原始消息
const message = "Hello Alephium";

// 转换为十六进制
const messageHex = Buffer.from(message).toString('hex');

const result = await HardwareSDK.alephiumSignMessage(connectId, deviceId, {
    path: "m/44'/1234'/0'/0/0",
    messageHex,  // 十六进制格式的 "Hello Alephium"
    messageType: "alephium"
});
```

#### 参数

[可选通用参数](../common-params)

* `path` - _必需_ `string | Array<number>` 最小长度为 3
* `messageHex` - _必需_ `string` 十六进制格式的待签名消息
* `messageType` - _可选_ `string` 消息签名算法类型（默认："alephium"）

#### 示例

```typescript
// 原始消息
const message = "Hello Alephium";

// 转换为十六进制
const messageHex = Buffer.from(message).toString('hex');

const response = await HardwareSDK.alephiumSignMessage(connectId, deviceId, {
    path: "m/44'/1234'/0'/0/0",
    messageHex,
    messageType: "alephium"
});
```

返回结果

```typescript
{
    success: true,
    payload: {
        signature: string;
        address: string;
    }
}
```

错误

```typescript
{
    success: false,
    payload: {
        error: string, // 错误信息
        code: number // 错误码
    }
}
```
