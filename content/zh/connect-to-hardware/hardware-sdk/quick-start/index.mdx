# 快速开始（Web，hd-common-connect-sdk）

本页面为您提供在浏览器中集成 OneKey 硬件钱包的简洁思维模型：初始化 SDK、发现设备、处理 UI 提示，以及发起首次调用。具体代码细节请查看子页面。

适用范围：使用 `@onekeyfe/hd-common-connect-sdk` 通过 WebUSB 连接的 Web（浏览器）应用。

1. 设置与初始化 — 安装 SDK，尽早初始化，必要时启用调试模式。
2. 设备发现 — 授权（WebUSB），使用 `searchDevices` 枚举设备，理解标识符（`connectId`、`device_id`），通过 `getFeatures` 解析 `device_id`。
3. PIN 码 — 监听 `UI_EVENT` 并通过 `uiResponse` 响应 `REQUEST_PIN`（推荐使用设备端输入）。查看 [事件配置](../api-reference/config-event)。
4. 密码短语 — 处理隐藏钱包的 Passphrase（设备端或软件输入，可选会话缓存）。
5. 第一个命令 — 使用 `connectId` + `device_id` 调用链 API（例如 `btcGetAddress`）。

下一步：为您的平台选择传输协议
