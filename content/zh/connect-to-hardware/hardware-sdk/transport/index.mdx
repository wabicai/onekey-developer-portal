# 传输协议

连接 OneKey 硬件设备的底层 USB 和蓝牙通信指南。

## 可用传输方式

### WebUSB

使用 WebUSB API 在网页浏览器中连接 OneKey 设备。

[查看 WebUSB 指南](web-usb)

### React Native BLE

React Native 移动应用的低功耗蓝牙通信。

[查看 React Native BLE 指南](react-native-ble)

### Common Connect

iOS、Android 和 Flutter 的平台特定连接实现。

[查看 Common Connect](native-ble)
