# Air‑Gap SDK API 参考

- `basic-api/` – 解码 QR 数据时使用的 `CryptoHDKey`、`CryptoCoinInfo` 和 `CryptoKeypath` 定义。
- `ethereum-and-evm/` – `EthSignRequest`、`EthSignature` 及其他 EVM 离线签名流程示例。

将此参考与快速入门配合使用以进行端到端集成：
- [Air‑Gap 快速入门](../quick-start)

提示：在实现扫描/解码之前先学习 `basic-api`，以确保正确解析。

Tron 说明：Tron Air‑Gap 辅助工具在我们的 QR 钱包 SDK 中，遵循相同的 UR 模式；使用方法请参阅 [Tron 快速入门](../quick-start/tron)。

依赖包
- 使用 Keystone 官方包：`@keystonehq/keystone-sdk`、`@keystonehq/bc-ur-registry*`
- UR 基础组件：`@ngraveio/bc-ur`
