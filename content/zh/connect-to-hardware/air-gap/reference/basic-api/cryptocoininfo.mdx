# CryptoCoinInfo

`CryptoCoinInfo` 类表示加密货币信息。

```typescript
export enum Type {
  bitcoin = 0,
  ethereum = 60,
}

export enum Network {
  mainnet,
  testnet,
}
```

### 参数

* `Type`：定义货币类型，例如 `bitcoin = 0`。
* `Network`：定义网络类型，例如 `mainnet` 和 `testnet`。



## URL 示例

```typescript
ur:crypto-coin-info/xxxxx
```
