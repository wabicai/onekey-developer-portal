# CryptoHDKey

`CryptoHDKey` 类表示分层确定性密钥信息。

这是 OneKey 硬件提供的指令，包含扩展公钥信息。

### 参数

* `isMaster`：`boolean` 是否为主密钥。
* `isPrivateKey`：`boolean` 是否为私钥。
* `key`：`Buffer` 密钥数据。
* `chainCode`：`Buffer` 链码。
* `useInfo`：[`CryptoCoinInfo`](cryptocoininfo) 使用信息。
* `origin`：[`CryptoKeypath`](cryptokeypath) 原始路径。
* `children`：[`CryptoKeypath`](cryptokeypath) 子路径。
* `parentFingerprint`：`Buffer` 父指纹。
* `name`：`string` 名称。可选
* `note`：`Note(string)` 备注。可选
  * 'account.standard'：BIP44 标准账户
  * "account.ledger_live"：Ledger Live 账户
  * "account.ledger_legacy"：Ledger Legacy 账户

## 解码流程（概要）

- 将 QR 帧聚合为 UR 对象（例如，使用 `@ngraveio/bc-ur` 的 `URDecoder`）。
- 检查 `ur.type === 'crypto-hdkey'` 并使用注册表辅助函数从 CBOR 字节解析 `CryptoHDKey`。
- 提取字段，如 name、note、chainCode、bip32 xpub、origin/children 路径和源指纹（xfp）。
