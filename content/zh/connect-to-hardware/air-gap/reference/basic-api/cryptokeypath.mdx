# CryptoKeypath

`CryptoKeypath` 类表示加密密钥路径。

### 参数

* `components`：`PathComponent[]` 路径组件数组，默认为空数组。
* `sourceFingerprint`：`Buffer` 源指纹，可选。
* `depth`：`number` 深度，可选。



### `PathComponent`

`PathComponent` 类表示加密密钥路径中的单个组件。

#### 参数

* `args`：Object
  * `index`：`number` 组件索引，可选。
  * `hardened`：`boolean` 是否为硬化派生。



## URL 示例

```typescript
ur:crypto-keypath/xxxxx
```
