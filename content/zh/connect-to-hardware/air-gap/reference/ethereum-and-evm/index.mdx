# Ethereum 和 EVM

要获取 ETH 账户，可以使用基础 API [`CryptoHDkey`](../basic-api/cryptohdkey) 来获取账户。



集成 ETH 时，可以使用 `EthSignRequest` 向硬件请求签名。目前支持：

* EIP1559 交易
* Legacy 交易
* EIP-712 TypedData
* 消息签名

有关如何构造相应 UR 数据的详细信息，请参阅：
- [EthSignRequest](ethsignrequest)




无论请求类型如何，结果都解析为 `EthSignature`：
- [EthSignature](ethsignature)
