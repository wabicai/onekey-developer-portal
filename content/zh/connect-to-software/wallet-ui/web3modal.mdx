# Web3Modal

## 文档

- 官方文档：https://docs.walletconnect.com/web3modal/about

## 安装

```bash
npm install @web3modal/wagmi wagmi@1.4.13 viem@1.21.4
# 或
yarn add @web3modal/wagmi wagmi@1.4.13 viem@1.21.4
```

## 配置 OneKey

将 OneKey 添加到 `featuredWalletIds` 和 `includeWalletIds`。

```javascript
createWeb3Modal({
  wagmiConfig,
  projectId,
  chains,
  featuredWalletIds: [
    // OneKey 钱包 ID，参见 https://explorer.walletconnect.com/onekey
    "1aedbcfc1f31aade56ca34c38b0a1607b41cccfa3de93c946ef3b4ba2dfab11c",
  ],
  includeWalletIds: [
    // OneKey 钱包 ID，参见 https://explorer.walletconnect.com/onekey
    "1aedbcfc1f31aade56ca34c38b0a1607b41cccfa3de93c946ef3b4ba2dfab11c",
  ],
});
```
