# getPublicKey

### 方法

```typescript
async function getPublicKey(): string
```

### 响应

返回十六进制格式的公钥

### 示例

```typescript
const provider = (window.$onekey && window.$onekey.nostr) || window.nostr;

const pubkey = async provider.getPublicKey()

```
