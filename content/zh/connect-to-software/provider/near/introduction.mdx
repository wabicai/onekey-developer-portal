# 介绍

[**OneKey**](https://www.onekey.so) 是一款钱包和浏览器扩展，可用于管理数字资产并访问 [Near](https://docs.near.org) 区块链上的去中心化应用。

它代表用户创建和管理私钥，允许他们存储资金和签署交易。

Web 应用可以与扩展钱包进行交互，例如请求执行交易的权限。
