# 发送交易

一旦 Web 应用连接到 OneKey，它就可以在获得用户许可的情况下代表用户发送交易。

要发送交易，Web 应用必须：

* 创建一个或多个未签名的交易。
* 让用户的 OneKey 钱包签名并提交到网络。

有关 Near 上交易的更多信息，建议查阅 `near-api-js` [文档](https://docs.near.org/docs/develop/front-end/near-api-js)以及官方 [Near 文档](https://docs.near.org/)。
