# 调试日志

你可以打印出 provider 和 OneKey 钱包之间的内部通信消息。此功能可能对你的应用调试和开发有用。

日志消息按 **jsBridge**、**providerBase** 和 **extInjected** 分组。你可以像这样打开或关闭它们，页面重新加载后生效：

```javascript
// 打开所有分组日志
localStorage.setItem('$$ONEKEY_DEBUG_LOGGER', '*');

// 关闭所有分组日志
localStorage.setItem('$$ONEKEY_DEBUG_LOGGER', '');

// 仅打开 `jsBridge` 分组
localStorage.setItem('$$ONEKEY_DEBUG_LOGGER', 'jsBridge');

// 同时打开 `jsBridge` 和 `providerBase` 分组
localStorage.setItem('$$ONEKEY_DEBUG_LOGGER', 'jsBridge,providerBase');

```
