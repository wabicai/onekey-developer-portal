# getNetwork

获取当前账户的网络

### 方法

```typescript
async function getNetwork(): string
```

### 响应

返回网络类型。可选值：`livenet`（主网）和 `testnet`（测试网）

### 示例

```typescript
const provider = (window.$onekey && window.$onekey.btc) || window.unisat;

const network = async provider.getNetwork()

```
