# getPublicKey

获取当前账户的公钥。

### 方法

```typescript
async function getPublicKey(): string
```

### 响应

返回公钥。

### 示例

```typescript
const provider = (window.$onekey && window.$onekey.btc) || window.unisat;

const publicKey = async provider.getPublicKey();

```
