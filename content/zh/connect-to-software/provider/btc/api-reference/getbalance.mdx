# getBalance

获取当前账户的余额。

### 方法

```typescript
async function getBalance(): BalanceInfo
```

### 响应

返回余额信息。

```typescript
type BalanceInfo = {
    confirmed: number;   // 已确认余额
    unconfirmed: number; // 未确认余额
    total: number        // 总余额
};
```

### 示例

```typescript
const provider = (window.$onekey && window.$onekey.btc) || window.unisat;

const balance = async provider.getBalance();

```
