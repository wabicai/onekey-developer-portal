# getAccounts

获取当前账户的地址

### 方法

```typescript
async function getAccounts(): string[]
```

### 示例

```typescript
const provider = (window.$onekey && window.$onekey.btc) || window.unisat;

const accounts = async provider.getAccounts()

```
