# pushTx

推送交易

### 方法

```typescript
async function pushTx(rawtx: string): string
```

### 参数

* `rawtx` — _必填_ `string` 要推送的原始交易

### 响应

`txid` — `string`

### 示例

```typescript
const provider = (window.$onekey && window.$onekey.btc) || window.unisat;

const rawtx = "010203"
const txid = async provider.pushTx(rawtx);

```
