# sendBitcoin

发送 BTC

### 方法

```typescript
async function sendBitcoin(
    toAddress: string,
    satoshis: number,
    options?: { feeRate: number }
): string
```

### 参数

* `toAddress` — _必填_ `string` 接收地址
* `satoshis` — _必填_ `number` 发送的聪数
* `options` — _可选_
  * `feeRate` — _必填_ `number` 网络费率

### 响应

`txid` — `string`

### 示例

```typescript
const provider = (window.$onekey && window.$onekey.btc) || window.unisat;

const address = "010203"
const txid = async provider.sendBitcoin(address, 1000);

```
