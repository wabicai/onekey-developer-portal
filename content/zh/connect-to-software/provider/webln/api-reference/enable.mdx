# enable

要开始交互，首先需要获取适当的权限。调用 `webln.enable()` 将提示用户允许使用浏览器的 Web 访问功能。之后，你可以自由调用任何其他 API 方法。

### 方法

```typescript
async function enable(): void
```



### 示例

```typescript
const provider = (window.$onekey && window.$onekey.webln) || window.webln;

async provider.enable()

```
