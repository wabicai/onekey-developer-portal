# WebLN

使用 OneKey 的 WebLN Provider 与支持闪电网络的应用交互。

## 快速链接

- [指南](guide)
- [API 参考](api-reference)
- [事件](event)

## 最小示例

```js
const provider = window?.$onekey?.webln || window?.webln
if (!provider) throw new Error('未检测到 OneKey WebLN Provider')

await provider.enable()
const info = await provider.getInfo()
```
