# JSON-RPC 方法参考

本参考文档详细介绍了 OneKey Ethereum Provider 支持的所有 JSON-RPC 方法。

---

## 账户方法

| 方法 | 描述 |
|------|------|
| [eth_requestAccounts](./eth_requestAccounts) | 请求用户连接账户 |
| [eth_accounts](./eth_accounts) | 获取已连接的账户 |
| [wallet_requestPermissions](./wallet_requestPermissions) | 请求 dApp 权限 |
| [wallet_getPermissions](./wallet_getPermissions) | 获取当前权限 |

---

## 交易方法

| 方法 | 描述 |
|------|------|
| [eth_sendTransaction](./eth_sendTransaction) | 发送交易 |
| [eth_call](./eth_call) | 执行只读合约调用 |
| [eth_estimateGas](./eth_estimateGas) | 估算 Gas 消耗 |
| [eth_getTransactionByHash](./eth_getTransactionByHash) | 通过哈希获取交易详情 |
| [eth_getTransactionReceipt](./eth_getTransactionReceipt) | 获取交易回执 |

---

## 签名方法

| 方法 | 描述 |
|------|------|
| [personal_sign](./personal_sign) | 签名消息 |
| [eth_signTypedData_v4](./eth_signTypedData_v4) | 签名结构化数据 (EIP-712) |

---

## 网络方法

| 方法 | 描述 |
|------|------|
| [eth_chainId](./eth_chainId) | 获取当前链 ID |
| [wallet_switchEthereumChain](./wallet_switchEthereumChain) | 切换网络 |
| [wallet_addEthereumChain](./wallet_addEthereumChain) | 添加新网络 |
| [wallet_watchAsset](./wallet_watchAsset) | 添加代币到钱包 |
