# getPublicKey

Get publicKey of current account.

### Method

```typescript
async function getPublicKey(): string
```

### Params

the publicKey.

### Example

```typescript
const provider = (window.$onekey &#x26;&#x26; window.$onekey.btc) || window.unisat;

const publicKey = async provider.getPublicKey();

```
