# Provider API

Direct access to OneKey's injected providers for all supported chains. No SDK required.

<Callout type="info">
For quick integration with built-in UI, consider using [RainbowKit](/en/connect-to-software/wallet-kits/rainbowkit) or [Web3Modal](/en/connect-to-software/wallet-kits/web3modal) instead.
</Callout>

## Provider Detection

### EIP-6963 (Recommended)

Modern multi-wallet detection standard:

```typescript
const providers: any[] = []

window.addEventListener('eip6963:announceProvider', (e: any) => {
  providers.push(e.detail)
})
window.dispatchEvent(new Event('eip6963:requestProvider'))

// Find OneKey by rdns
const onekey = providers.find(p => p.info.rdns === 'so.onekey.app.wallet')
```

### Direct Access

```typescript
// EVM
const evmProvider = window.$onekey?.ethereum

// Non-EVM (replace 'solana' with your chain)
const solanaProvider = window.$onekey?.solana
```
