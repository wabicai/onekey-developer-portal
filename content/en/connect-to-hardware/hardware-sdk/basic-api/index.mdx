# Basic API

Core SDK methods for device discovery, session management, and event handling.

| Method | Description |
|--------|-------------|
| [Search Devices](./search-devices) | Scan and discover connected OneKey hardware devices |
| [Get Features](./get-features) | Retrieve device capabilities and current state |
| [Get Passphrase State](./get-passphrase-state) | Get the current passphrase wallet state |
| [Response UI Event](./response-ui-event) | Handle UI events from device interactions |
| [Cancel Request](./cancel-request) | Cancel an ongoing device operation |
