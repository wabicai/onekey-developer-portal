# CryptoCoinInfo

The `CryptoCoinInfo` class represents cryptocurrency information.

## Types

```typescript
export enum Type {

  ethereum = 60,

export enum Network {
  mainnet,
  testnet,
}
```

| Parameter | Type | Description |
|-----------|------|-------------|
| `Type` | `enum` | Currency type (e.g., `bitcoin = 0`, `ethereum = 60`) |
| `Network` | `enum` | Network type (`mainnet` or `testnet`) |
